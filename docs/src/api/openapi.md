# OpenAPI Explorer

Live interactive reference for the platform API. The explorer below is served
from the platform's live docs at `http://localhost:8080/docs`.

!!! note
    Platform must be running for the explorer to load. Host runner has its own
    schema at `http://localhost:8090/openapi.json`.

Useful links:

- [Platform Swagger UI](http://localhost:8080/docs)
- [Platform OpenAPI schema](http://localhost:8080/openapi.json)
- [Host runner OpenAPI schema](http://localhost:8090/openapi.json)
